Demo: Project Planning for YouTrack

Reconstruct past project schedule from YouTrack activity log, and compute future schedule from YouTrack issue data plus a list of available contributors.

This web app is deliberately bare-bones to demonstrate the API provided by this package. All data is retrieved through the YouTrack REST API. Authorization happens through OAuth 2.0. This web app never comes in touch with your password.

For an InCloud instance, this is of form https://<name>.myjetbrains.com/.
The YouTrack service ID is available via Server Settings > Hub Integration.
If you are not logged into YouTrack yet, this will take you to the YouTrack login page. Once logged in, you will be redirected back here. Please note: The URI of this web app (that is, “”) needs to be registered in the Hub Settings under “Redirect URIs”. The URI also needs to be added to Server Settings > Global Settings under “Allowed origins”.
